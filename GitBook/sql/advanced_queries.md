# Almennt form SELECT skipunar

```sql
SELECT <dálkar eða útreikningur>
FROM <tafla>
WHERE <skilyrði>
GROUP BY <dálkar>
HAVING <skilyrði>
ORDER BY <dálkar>
    LIMIT <tala>
OFFSET <tala>;
```
